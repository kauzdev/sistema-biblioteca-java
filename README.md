# Sistema de Gerenciamento de Biblioteca em Java

Projeto desenvolvido em Java utilizando os conceitos de Programação Orientada a Objetos (POO).

## Funcionalidades

* Cadastro de livros
* Listagem de livros cadastrados
* Empréstimo de livros
* Devolução de livros
* Controle de disponibilidade dos exemplares

## Tecnologias Utilizadas

* Java
* IntelliJ IDEA
* Git
* GitHub

## Estrutura do Projeto

```text
src
├── model
│   └── Livro.java
├── service
│   └── Biblioteca.java
├── utils
│   └── Menu.java
└── Main.java
```

## Conceitos Aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento
* Classes e Objetos
* Construtores
* Métodos
* ArrayList
* Organização em pacotes
* Separação de responsabilidades
